Cyber Security Roadmap

Complete guide to learning cybersecurity from beginner to advanced

Basic Level

Computer Networks

Understanding network concepts like TCP/IP, HTTP/HTTPS, DNS, SSL, etc.

Networks Lectures (YouTube) Computer Networking Notes Coursera - Computer Networking Khan Academy - Computers and Internet

Operating Systems

Understanding OS concepts like process management, memory, file systems, security.

OSTEP Book OS Lectures (YouTube) Coursera - Operating Systems edX - Introduction to Linux

Introduction to encryption, digital signatures, concepts like AES, RSA, Hash Functions.

Cryptography - Khan Academy Practical Cryptography for Developers Coursera - Cryptography Udacity - Applied Cryptography

Git & GitHub

Git is a version control system for tracking changes in source code. GitHub is a platform for hosting repositories and collaborating on projects.

Understanding common web application attacks like XSS, CSRF, SQL Injection, etc.

Cisco Network Security SANS Network Security Coursera - Network Security Cybrary - Network Security

Techniques for testing security and identifying vulnerabilities.

Metasploit Unleashed TryHackMe HackTheBox EC-Council - Certified Ethical Hacker

Common tools for security testing and network analysis.

Nmap Documentation Wireshark Documentation Burp Suite Documentation Udemy - Nmap for Beginners IppSec - Security Tutorials (YouTube)

Reverse Engineering

Analyzing software to understand how it works and find vulnerabilities.

Begin.RE Awesome Reversing Resources Coursera - Reverse Engineering Udemy - Reverse Engineering and Malware Analysis
