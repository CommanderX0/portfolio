Data Science Roadmap

Complete guide to learning data science from beginner to advanced

Basic Level

Mathematics for Data Science

Essential mathematics like linear algebra, calculus, statistics, probability required to understand data science algorithms.

Linear Algebra - Khan Academy Statistics & Probability Coursera - Mathematics for Machine Learning edX - Linear Algebra

Programming (Python)

Python is the primary language for data science, with libraries like NumPy, Pandas, Matplotlib.

Python Getting Started NumPy Quickstart Pandas 10 Minutes Guide Coursera - Python for Everybody Matplotlib Tutorials

Data Manipulation & Analysis

Skills for cleaning, transforming, and analyzing data using libraries like Pandas and NumPy.

Pandas 10 Minutes Guide NumPy Quickstart Coursera - Data Analysis with Python Kaggle - Pandas Course

Git & GitHub

Git is a version control system for tracking changes in source code. GitHub is a platform for hosting repositories and collaborating on projects.

Official Git Documentation GitHub Docs Atlassian - Git Tutorials Udacity - Version Control with Git

Beyond Basic

Statistical Analysis

Understanding statistical concepts and methods for data analysis, hypothesis testing, etc.

Statistics & Probability Hypothesis Testing Coursera - Statistics for Data Science Udacity - Intro to Statistics

Data Visualization

Creating effective visualizations using libraries like Matplotlib, Seaborn, Plotly, etc.

Matplotlib Tutorials Seaborn Tutorial Plotly Python Documentation Coursera - Data Visualization with Python Kaggle - Data Visualization Course

Machine Learning Fundamentals

Introduction to ML concepts like supervised/unsupervised learning, evaluation, etc.

ML Course by Andrew Ng Scikit-learn Guide Kaggle - Intro to Machine Learning fast.ai - Practical Deep Learning

Data Cleaning & Preprocessing

Essential steps to prepare data before analysis or modeling.

Pandas Missing Data Scikit-learn Preprocessing Kaggle - Data Cleaning Coursera - Data Preprocessing

Advanced Level

Deep Learning

Introduction to deep neural networks, TensorFlow/PyTorch, etc.

Deep Learning Specialization PyTorch Tutorials TensorFlow Tutorials fast.ai - Deep Learning for Coders Coursera - Deep Learning Specialization

Big Data Technologies (Spark, Hadoop)

Handling and processing large datasets using distributed computing frameworks.

Apache Spark Documentation Apache Hadoop Documentation Coursera - Introduction to Big Data Udacity - Spark and Hadoop

Cloud Platforms (AWS, GCP, Azure)

Using cloud services for data storage, processing, and machine learning.

AWS SageMaker Google AI Platform Azure Machine Learning Coursera - AWS Machine Learning Udacity - Azure ML Engineer

Production & MLOps

Deploying models to production and managing the ML lifecycle.

MLflow Documentation Kubeflow Documentation Coursera - Introduction to MLOps Udacity - Machine Learning DevOps
